## üìù Convert Markdown to PDF

### Option 1: Online (Easiest)
1. Open `theoretical_answers.md` in any text editor
2. Copy all content (Cmd+A, Cmd+C)
3. Go to https://www.markdowntopdf.com or https://md2pdf.netlify.app
4. Paste and click "Convert"
5. **Add your name** to the PDF
6. Download as `theoretical_answers.pdf`

### Option 2: Using Pandoc (If installed)
```bash
./convert_to_pdf.sh
```

### Option 3: VS Code
1. Install extension: "Markdown PDF" by yzane
2. Open `theoretical_answers.md`
3. Press `Cmd+Shift+P`
4. Type "Markdown PDF: Export (pdf)"
5. **Add your name** to the PDF
